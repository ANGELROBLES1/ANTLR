Analizador Sintáctico en Python
Descripción

Este proyecto contiene tres programas en Python que implementan analizadores sintácticos descendentes recursivos para diferentes gramáticas.

Cada archivo corresponde a un ejercicio y permite evaluar si una cadena es aceptada o no según las reglas definidas.

Archivos
Ejercicio1.py
Ejercicio2.py
Ejercicio3.py

Ejecutar el archivo que se desee
python Ejercicio1.py
python Ejercicio2.py
python Ejercicio3.py
Entrada

Las cadenas que se evalúan están definidas dentro del código en una lista llamada tokens.

Ejemplo:

tokens = ["dos", "tres", "$"]
Salida

El programa mostrará en consola:

"Cadena aceptada" si la cadena cumple la gramática
"Error sintáctico" si la cadena no es válida
